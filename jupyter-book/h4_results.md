# H4. MVTec AD — 산업 이미지 이상탐지 결과

## 데이터셋

| 항목 | 내용 |
|------|------|
| 데이터셋 | MVTec Anomaly Detection (MVTec AD) |
| 분석 카테고리 | bottle |
| 학습 데이터 | 정상(good) 이미지만 14장 |
| 테스트 데이터 | 83장 (정상 20장 + 결함 63장) |
| 이미지 해상도 | 700~1024px → **256×256** 리사이즈 |

## 결함 유형

| 결함 유형 | 테스트 샘플 수 |
|-----------|---------------|
| broken_large | 20 |
| broken_small | 22 |
| contamination | 21 |
| good (정상) | 20 |
| **총계** | **83** |

## 모델 1: Autoencoder

### 아키텍처

| 구간 | 레이어 | 크기 변화 |
|------|--------|-----------|
| 인코더 | Conv2D(64, strides=2) + BN | 256 → 128 |
| 인코더 | Conv2D(128, strides=2) + BN | 128 → 64 |
| 인코더 | Conv2D(256, strides=2) + BN | 64 → 32 |
| 인코더 | Conv2D(512, strides=2) + BN | 32 → 16 |
| 잠재공간 | Flatten | 16×16×512 → 131,072 |
| 디코더 | Conv2DTranspose(512~64, strides=2) + BN | 16 → 256 |
| 출력 | Conv2D(3, sigmoid) | 256×256×3 |
| **파라미터 수** | | **5,468,931** |

### 학습 결과 (50 Epoch)

| Epoch | Train Loss (MSE) | Train MAE | Val Loss | Val MAE |
|-------|-------------------|-----------|----------|---------|
| 1 | 0.1929 | 0.3694 | 0.1292 | 0.3267 |
| 10 | 0.0417 | 0.1530 | 0.1291 | 0.3263 |
| 30 | 0.0118 | 0.0876 | 0.1277 | 0.3245 |
| 50 | 0.0077 | 0.0714 | 0.1254 | 0.3221 |

### 이상 탐지 결과

| 지표 | 값 |
|------|-----|
| 임계값 (mean + 2×std) | 0.1303 |
| 학습 오차 mean | 0.1276 |
| **AUROC** | **0.2865** |

> AUROC가 낮은 이유: 학습 정상 이미지가 14장으로 매우 적어, 결함 이미지도 정상처럼 재구성되는 경향이 있음.

## 모델 2: PatchCore (kNN 기반)

### 파이프라인

| 단계 | 내용 |
|------|------|
| 백본 | MobileNetV2 (ImageNet 사전학습, 가중치 동결) |
| 특징 레이어 | "out_relu" → 8×8×1280 |
| Memory Bank | 14장 × 81,920차원 = (14, 81920) |
| 이상 점수 | kNN (k=3) 평균 L2 거리 |
| 임계값 | mean + 2×std = 69.6079 |

### 이상 탐지 결과

| 지표 | 값 |
|------|-----|
| **AUROC** | **0.9913** |
| **F1 Score** | **0.6551** |

## 두 방법 비교

| 지표 | Autoencoder | PatchCore |
|------|-------------|-----------|
| 방식 | 재구성 오차 (MSE) | 특징 거리 (kNN) |
| 임계값 | 0.1303 | 69.6079 |
| **AUROC** | 0.2865 | **0.9913** |
| **F1 Score** | — | **0.6551** |

> **PatchCore가 압도적으로 우수** (AUROC 0.99 vs 0.29). 사전학습된 MobileNetV2 특징이 정상/결함 구분에 매우 효과적.

## 결함 로컬라이제이션

- 재구성 오차를 픽셀 단위로 계산하여 결함 위치 시각화
- percentile(95) 임계값으로 오차 상위 5%만 결함으로 이진화
- GT 마스크(ground_truth)와 비교하여 탐지 정확도 확인

## 핵심 인사이트

- **학습 데이터가 14장뿐인 극소样本 환경**에서도 PatchCore는 AUROC 0.99 달성
- Autoencoder는 데이터가 적을 때 결함도 정상처럼 재구성하는 한계가 명확히 드러남
- PatchCore의 핵심: 사전학습 특징을 활용하므로 학습 데이터가 적어도 강력한 성능
- 결함 로컬라이제이션으로 "어디가 이상한지"까지 픽셀 수준으로 파악 가능
