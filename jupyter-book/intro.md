## 이 Jupyter Book 사용법

### Colab에서 실행하기

```{tip}
이 책의 모든 노트북은 Google Colab에서 직접 실행할 수 있습니다.
```

각 노트북 페이지의 코드는 책 내에서 읽기 전용으로 표시됩니다. 직접 실행하려면 아래 방법 중 하나를 선택하세요.

1. **GitHub에서 다운로드 후 Colab에 업로드**
   - GitHub 저장소에서 `.ipynb` 파일의 **Raw** 버튼을 우클릭하여 **링크 주소 복사**
   - [Google Colab](https://colab.research.google.com) 접속 → **파일 > 노트 업로드** 또는 **GitHub 탭에서 URL 붙여넣기**
   - 업로드된 노트북을 바로 실행

2. **브라우저 확장 프로그램 활용**
   - Chrome 확장 프로그램 **"Open in Colab"** 을 설치하면, GitHub의 `.ipynb` 파일 페이지에서 버튼 한 번으로 Colab으로 열 수 있습니다.

3. **Colab의 GitHub 탭 사용**
   - Colab 열기 대화상자에서 **GitHub** 탭 선택 → 저장소 URL 입력 → 원하는 노트북 선택

### 섹션 간 탐색하기

```{note}
효율적인 학습을 위해 아래 탐색 팁을 활용하세요.
```

- **사이드바 활용**: 왼쪽 사이드바의 각 섹션을 **우클릭 → "새 탭에서 열기"** 하면 여러 섹션을 오가며 학습할 수 있습니다.
- **브라우저 탭 활용**: 강의 노트 페이지와 코드 실습 노트북을 각각 다른 탭에 열어두면 참고하기 편합니다.
- **검색 기능**: 상단의 돋보기 아이콘을 클릭하면 전체 교재에서 키워드를 검색할 수 있습니다.
- **목차 바로가기**: 사이드바 상단의 목차 아이콘으로 현재 페이지의 섹션 구조를 확인할 수 있습니다.

### 코드 실습 팁

```{warning}
책 내의 코드 셀은 **읽기 전용**입니다. 코드를 직접 실행하려면 반드시 노트북을 다운로드하세요.
```

- **코드 다운로드**: 각 실습 페이지의 노트북 파일(`.ipynb`)을 다운로드하여 Colab 또는 로컬 환경에서 실행하세요.
- **Claude Code 프롬프트 활용**: 각 섹션에는 Claude Code 프롬프트 예시가 포함되어 있습니다. 프롬프트를 **복사해서 Claude Code에 붙여넣기** 하면 AI가 코드를 작성하고 설명해 줍니다.
- **실습 시작 코드(Starter Code)**: 프롬프트 템플릿을 복사하여 Colab이나 로컬 환경에서 바로 실습을 시작할 수 있습니다.
