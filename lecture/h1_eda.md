## 4. 주파수 분석 (FFT)

### 대표 샘플 지배 주파수 (상위 3개)

| 조건 | 축 | 1차 주파수 | 2차 주파수 | 3차 주파수 |
|------|-----|-----------|-----------|-----------|
| Normal | acc_x | 1,025Hz | 1,494Hz | 1,611Hz |
| Normal | acc_y | 1,523Hz | 2,441Hz | 1,416Hz |
| Chatter | acc_x | **3,984Hz** | 3,994Hz | 4,033Hz |
| Chatter | acc_y | **3,984Hz** | 3,994Hz | 3,975Hz |
| Wear | acc_x | 2,061Hz | 1,777Hz | 4,912Hz |
| Wear | acc_y | 4,414Hz | 1,543Hz | 1,553Hz |

**핵심 관찰**:
- **Normal**: 1~2.5kHz 대역에 주파수 성분 분포 → 정상 절삭 진동
- **Chatter**: ~4kHz 대역에 에너지 집중 → 고주파 채터링 진동. Nyquist 주파수(5kHz) 근처에 집중된 것은 고주파 불안정 진동의 특징
- **Wear**: 1.5~4.9kHz 대역에 넓게 분포 → 마모로 인한 불규칙 진동 패턴
- 세 가지 조건이 주파수 대역에서 뚜렷하게 구분됨 → **FFT/Wavelet 기반 분류가 효과적**일 것

## 6. 클래스 불균형 분석

### 3-class 분포

```
Wear:    2,332,650 (77.9%) ████████████████████████████████████████
Normal:    360,207 (12.0%) ██████
Chatter:   301,450 (10.1%) █████
```

### 2-class (정상 vs 이상) 분포

```
Anomaly: 2,634,100 (88.0%) ████████████████████████████████████████████████████████████████████████
Normal:    360,207 (12.0%) ████████████
```

**세션 2와의 연결**:
- 이 데이터셋은 전형적인 불량 탐지 시나리오와 **반대**입니다. 보통은 정상이 99%이고 불량이 1%인데, 여기서는 이상(wear+chatter)이 88%입니다.
- 만약 "정상 vs 이상" 이진 분류를 하면, 정상이 소수 클래스(12%)가 됩니다.
- 실제 현장에서는 정상 데이터가 대부분이고 이상이 드문데, 이 데이터셋은 실험 환경에서 이상 조건을 의도적으로 많이 수집한 것으로 보입니다.

## 7. 실습 파이프라인 적용을 위한 인사이트

### 섹션 1 (노이즈 제거) 연결
- Normal 데이터의 std가 5~40으로 낮고, Chatter/Wear의 std가 100~900으로 높음
- Wavelet 노이즈 제거 시 고주파 대역(채터링 ~4kHz)을 threshold로 걸러내면, 정상 신호(1~2kHz)만 남을 것
- 이 데이터는 시뮬레이션이 아닌 **실제 센서 데이터**이므로, pywt 파이프라인을 그대로 적용 가능

### 섹션 2 (불균형 처리) 연결
- 클래스 불균형이 존재하지만 방향이 반대 (이상이 다수)
- SMOTE를 적용할 때는 **정상 클래스를 소수 클래스로 간주**하고 처리해야 함
- F1-Score보다는 클래스별 Recall을 개별적으로 확인하는 게 중요

### 섹션 3 (이상탐지 파이프라인) 연결
- RMS, Kurtosis, Peak Value 특징이 조건 간에 뚜렷한 차이를 보임
- 특히 RMS: Normal(28) < Wear(220) < Chatter(363) → Isolation Forest가 효과적으로 작동할 것
- FFT 주파수 대역 차이를 활용하면 Wavelet 기반 특징 추출도 효과적
