## 1. 데이터 개요

- **파일 수**: **12개 CSV** (라벨별 분리) + 부속 CAD 파일 `Facing_18x18.SLDPRT` 1개(EDA 제외)
- **컬럼**: `acc_x`, `acc_y`, `acc_z` — 3축 가속도(accelerometer)
- **총 샘플 수**: **2,994,307 행**
- **결측치**: 모든 라벨에서 0건
- **단위**: `acc_z` 평균이 모든 라벨에서 ~1,005로 안정 → **mg(milli‑g)** 로 추정(중력 1g ≈ 1000mg, z축이 위를 향하도록 거치). 헤더에 단위 표기는 없음.
- **시간/샘플링 정보 부재**: 타임스탬프·샘플링 주파수(fs) 컬럼이 **없음**. 따라서 주파수 분석은 **정규화 주파수(cycles/sample)** 로만 정직하게 표기하며, 절대 Hz 값은 fs가 별도로 확인되기 전까지 산출하지 않는다(7절 참고).

## 5. 분포 형태 (왜도·첨도)

- **라벨 `01`(정상)의 숨은 스파이크**: σ는 작지만 첨도가 acc_x **2900**, acc_y **2328**, acc_z **965**로 극단적. acc_x min/max = **-1,299.5 / +720.3 mg**, acc_z max **2,476 mg**(중력의 2.5배). → 평소 정지 + **드물게 강한 충격/탭**이 섞인 신호. 단순 σ만 보면 놓치는 패턴.
- 라벨 `00`, `02`는 첨도 < 1로 거의 정규분포(균질한 미세 진동).
- 고동역군은 대부분 |skew| < 0.3로 대칭. 단 라벨 `23`·`25`의 acc_y 첨도(5.5·5.7) → 간헐적 큰 y축 이벤트.

## 7. 주파수 분석 (정규화 주파수 기반)

데이터에 fs가 없으므로 **정규화 주파수(cycles/sample, 나이퀴스트=0.5)** 로 지배 성분을 산출했다(대표 라벨 4096 샘플, DC 제거 후 rFFT 상위 피크).

| 조건 (대표) | 축 | 지배 정규화 주파수 (cycles/sample) |
|---|---|---|
| Normal (label_00) | acc_x | 0.084, 0.096, 0.113 |
| Chatter (label_11) | acc_x | **0.398**, 0.106, 0.399 |
| Chatter (label_11) | acc_y | **0.398**, 0.399, 0.398 |
| Wear (label_20) | acc_x | 0.155, 0.143, 0.455 |
| Wear (label_20) | acc_y | 0.441, 0.155, 0.441 |

**핵심 관찰**
- **Normal**: 저주파(≈0.08~0.11) 에 에너지 집중 → 느리고 안정적인 진동.
- **Chatter**: **≈0.40(나이퀴스트의 약 80%)** 의 고주파에 강하게 집중 → 고주파 떨림(채터링)의 전형.
- **Wear**: 0.14~0.46 의 넓은 대역에 분포 → 불규칙한 마모 진동.
- 세 조건이 주파수 영역에서도 뚜렷이 구분 → FFT/Wavelet 기반 특징이 효과적.
- ⚠️ **주의**: fs가 확인되면 `Hz = 정규화주파수 × fs` 로 환산 가능하다. 예컨대 fs가 10kHz라면 0.398 → 약 3,980Hz가 되지만, **fs는 본 데이터에 포함되어 있지 않으므로 절대 Hz 값을 단정하지 않는다**(이전 버전의 "≈4kHz" 표기는 가정된 fs에 의존한 값이었음).

## 8. 분류·파이프라인 인사이트

| 그룹 | 라벨 | acc_z σ 범위 | 해석 |
|---|---|---|---|
| 0x 정상 | 00,01,02 | 5~40 | 정지/미세 진동 (01은 간헐 스파이크) |
| 1x 채터링 | 11,12 | 550~900 | 가장 큰 변동성, 고주파 떨림·축 결합 |
| 2x 마모 | 20~26 | 290~460 | 중간 강도, 넓은 주파수 분포 |

- **3-조건 상위 분류**(정상/채터링/마모)는 σz·RMS 룰만으로도 거의 분리 가능.
- **세부 라벨 분류**(20 vs 21 등)는 평균·σ만으론 어려움 → 시간·주파수 도메인 피처(FFT, zero-crossing, autocorrelation, jerk, Crest Factor 등) 필요.
- **섹션 1(노이즈 제거)**: 정상 σ가 5~40으로 낮고 채터링·마모가 100~900으로 높음 → Wavelet으로 고주파 성분을 정리하면 추세 신호가 부각.
- **섹션 2(불균형)**: 정상이 소수(12%) → SMOTE 시 정상을 소수 클래스로 간주, F1뿐 아니라 클래스별 Recall을 개별 확인.
- **섹션 3(이상탐지)**: RMS·Kurtosis·Crest Factor가 조건 간 뚜렷한 차이 → 정상만으로 학습하는 Isolation Forest가 효과적으로 작동.

## 9. 데이터 품질 이슈 및 권고

1. **타임스탬프/샘플링 레이트 부재** → 절대 주파수(Hz) 해석 불가. 메타데이터로 fs 명시 필요.
2. **라벨 `00`의 49% 중복**은 정지 상태 자연 현상 → 제거 금지.
3. **라벨 `24` 표본 부족**(40k, 다른 2x의 약 1/10) → 학습 시 클래스 가중치/oversampling 고려.
4. **라벨 `01`의 극단 outlier**(acc_x -1,299mg 등) → 측정 오류인지 의도된 충격인지 도메인 확인 필요.
5. **단위 표기 부재**(mg 추정) → 데이터 사전 작성 권고.
